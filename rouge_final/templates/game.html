🔄 Auto-update — how it works

  • The game launcher calls /api/version on every startup.
  • If the server version differs from the locally saved version, the launcher downloads the new build automatically.
  • The updated game runs on the very next launch — no manual steps needed.
  • Admins push new builds from the Admin panel — all connected clients update silently.
  • With Force Update enabled, players are blocked from playing until they have re-downloaded.
