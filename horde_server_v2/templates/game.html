How does browser play work?

The game uses Pygbag to compile spill.py into WebAssembly. The first load takes 10–30 seconds while the runtime downloads (~5 MB). After that it runs at full speed. Your progress is saved in localStorage and syncs to the server leaderboard if you're signed in.

⚠ To enable browser play:

Run pip install pygbag then pygbag --build spill.py to generate the build/web/ folder, then copy its contents to horde_server/static/game/. The server will serve it automatically.
